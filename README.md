# One key
It's a 2 player fighting game where you have to write your combos in morse code.

## How to play
   - A short tap is a "0", a short press is a "1" and a long press is a " "
   - multiple actions can be done at once, for example "010 0001" would do a jump then a right attack

## Moves
- MOVEMENT:
    - walk left: "10"
    - walk right: "01"
    - jump: "010"
    - dash left: "10110"
    - dash right: "01101"
    - high jump: "01110"
- ATTACKS:
    - left attack: "1000"
    - right attack: "0001"
    - down attack: "0110"
    - up attack: "1001"
